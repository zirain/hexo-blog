## Windows

### 启用go module
```
1. 右键 我的电脑 -> 属性 -> 高级系统设置 -> 环境变量
2. 在 “[你的用户名]的用户变量” 中点击 ”新建“ 按钮
3. 在 “变量名” 输入框并新增 “GO111MODULE”
4. 在对应的 “变量值” 输入框中新增 “on”
5. 最后点击 “确定” 按钮保存设置
```

### 配置GOPROXY
```
1. 右键 我的电脑 -> 属性 -> 高级系统设置 -> 环境变量
2. 在 “[你的用户名]的用户变量” 中点击 ”新建“ 按钮
3. 在 “变量名” 输入框并新增 “GOPROXY”
4. 在对应的 “变量值” 输入框中新增 “https://mirrors.aliyun.com/goproxy/”
5. 最后点击 “确定” 按钮保存设置
```
